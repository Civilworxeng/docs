## Stages Overview (SB001 - Civilworx Nova Court)

#### Summary:

Each Civil3D file type will have stages associated to it, this gives you a starting point at different "Stages" for a project setup. This gives the user to intentionally start and focus on a specific objective for training.

#### How to Implement:

Work in progress

<Tip>
  Each stage next in the subsequential order is in addition the previous stage numbers
</Tip>
