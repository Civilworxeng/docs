#### Strategy:

The intent for having a standardized naming convention is about conveying the most information so that any users can identify clear intent

### Windows Root Directory Limitations:

File naming comes at a cost as the attached folder directory attaches itself to the file name, some filetypes can be rendered as non readable if the total character limit exceeds 255-260.\
\
Pipe Network Catalogs are a high risk example of a common Civil3D File type that can become non readable or the files will not be able to copy over.
