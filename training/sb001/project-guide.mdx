    Verify site geometry consistency, including centerline alignment, parallel offsets, intersection connectivity, correct layer assignment, and all linework/block elevations set to 0.

      1. **Verify drawing settings is in international feet**\
         Toolspace → Settings → Right Click on drawing name → Drawing Settings
      2. **Check geometry parallel linework**\

      3. **Check geometry parallel linework**\
         Verify the linework and parallel  features run consistently along each entity intentionally.
      4. **Verify offsets**\
         Confirm  offsets are consistent and intentional across the site.
      5. **Check intersections**\
         Ensure linework meet exactly at crossing intersections.
      6. **Polylines joined or broken **\
         Ensure that all polylines are joined intentionally and broken intentionally for quick extraction
      7. **Review layers**\
         Use `LAYISO` to confirm site features are on the correct layers.
      8. **Confirm elevations**\
         Set all 2D linework and blocks to elevation `0`.
