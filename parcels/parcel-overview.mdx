### Summary:

Parcels should be used to label enclosed boundaries and can improve production workflows by XREFing in the parcel drawing to where you can update your label values in a centralized location

### Strategy:

Were able to utilize the functionality of XREF labeling drainage area maps basin labels or labeling lots, blocks and tracts/outlots efficiently. With the ability to label through XREFs we have the ability to manually adjusting the label style for each viewport or annotative scale changes.
